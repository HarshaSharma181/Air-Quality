# 🌍 Air Quality Data Visualization Analysis

## 📌 Project Overview

This project focuses on analyzing and visualizing air quality data collected across different locations in New York City. Using Python data analysis and visualization libraries, the project identifies pollution trends, compares pollution levels across regions, and highlights the most and least polluted areas.

The analysis helps understand environmental conditions and provides insights that can support public health initiatives and environmental policy decisions.

---

## 🎯 Objectives

* Analyze air quality data by location.
* Identify the most and least polluted places.
* Explore pollution patterns using various visualization techniques.
* Study the distribution of pollution levels and detect potential outliers.
* Generate meaningful insights for environmental monitoring and decision-making.

---

## 📊 Dataset Information

**Source:** NYC Open Data / Data.gov

The dataset contains:

* 18,025 records
* 11 attributes
* Air quality indicators such as PM2.5, SO₂ emissions, and related environmental measurements.

### Key Fields

| Field         | Description                 |
| ------------- | --------------------------- |
| Unique ID     | Unique record identifier    |
| Indicator ID  | Air quality indicator code  |
| Name          | Pollutant type              |
| Measure       | Unit of measurement         |
| Place         | Location of data collection |
| Time Period   | Year/season of measurement  |
| Data Value    | Recorded pollution value    |
| Geo Type Name | Geographic category         |
| Geo Join ID   | Geographic identifier       |

## 📈 Visualizations Implemented

### 1️⃣ Pie Chart

Shows the distribution of records across locations with smaller categories grouped as "Others".

### 2️⃣ Bar Graph

Displays top locations with the highest average pollution levels.

### 3️⃣ Line Chart

Illustrates pollution trends among highly polluted areas.

### 4️⃣ Scatter Plot

Visualizes pollution value distribution and variability across locations.

### 5️⃣ Histogram

Shows the frequency distribution of pollution measurements.

### 6️⃣ Horizontal Bar Graph

Provides a clearer comparison of pollution levels across locations.

### 7️⃣ Most & Least Polluted Places

Highlights environmental hotspots and cleaner regions.

---

## 🔍 Key Findings

### Most Polluted Areas

* High Bridge – Morrisania
* Hunts Point – Mott Haven
* Crotona – Tremont
* East Harlem
* Central Harlem – Morningside Heights

### Least Polluted Areas

* Tottenville and Great Kills
* South Beach – Tottenville
* Rockaway and Broad Channel
* Southern Staten Island
* Northern Staten Island

### Additional Insights

* A few locations consistently exhibit significantly higher pollution levels.
* Most pollution values fall within moderate ranges.
* Large differences exist between the most and least polluted regions.
* Pollution monitoring should prioritize identified hotspot areas.
